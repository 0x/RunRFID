# RunRFID


## Build
- ```cd RunRFID```
- ```mkdir build```
- ```cd build```
- ```cmake ..```
- ```make```

## Run
- ```./RunRFID <device_name>```

### libserialport
https://sigrok.org/api/libserialport/unstable/index.html
- Just install from apt: ```sudo apt install libserialport``` or ```sudo apt install libserialport-dev```
